# 국내 뉴스 알림 기능 개발 완료

## 프로젝트 개요

네이버 뉴스를 검색하여 매일 아침 대한민국의 주요 뉴스를 브리핑해주는 디스코드 봇 기능입니다.

## 최종 구현 사항

1.  **명령어**: `/news` (관리자 전용)
2.  **출력 형식**:
    - **핵심 뉴스 3개** 선정
    - **제목**: `1. [제목]`
    - **내용**: `- 요약`, `- 링크` (불렛 형식으로 가독성 확보)
3.  **기술적 특징**:
    - **No Parsing**: AI가 처음부터 마크다운 완제품을 생성하도록 프롬프트 엔지니어링 수행 (파싱 에러 원천 차단)
    - **입마개 해제**: 4,000 토큰 확보로 긴 뉴스 요약도 짤림 없이 출력
    - **배포 안전성**: `dist/` 폴더 기반의 클린 배포 파이프라인 구축

## 사용 방법

- DM 또는 서버 채널에서 `/news` 입력 시 즉시 발송됩니다.
- 스케줄러에 의해 매일 아침(08:00) 등 정해진 시간에 자동 발송됩니다.
