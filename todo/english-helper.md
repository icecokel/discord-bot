## 영어 학습 도우미

### 목적

- 매일 영어 학습을 위해 일상적인 영어 문장을 알려줌

### 기능

- 매일 점심 13시 KST 기준 discord-bot이 들어가있는 채널에 그날 영어 문장을 발송한다.

1. 문장 생성

- 일상, 비즈니스, 여행 등 카테고리를 선별 후 ai 를 통해 문장을 생성
- 매일 요일 별 쿠션 멘트를 지정하고, 쿠션 멘트 와 함께 문장을 발송한다.

2. admin 기능

- admin으로 지정된 사용자가 특정 커맨드를 발송하면, admin dm으로 그 사용자에게만 1.의 내용을 dm으로 송부한다. (스케줄링과 상관 없는 테스트를 위함)

### 불필요한 기능

- 유저의 설정 여부는 전혀 상관없다.
- 캐시가 필요하지 않다.
- 정적 파일을 읽고 쓸 필요가 없다.
